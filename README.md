# CODSOFT Python Programming Internship

This repository contains the tasks completed for the CODSOFT Python Programming Internship.

## Tasks

### 1. Password Generator
- Generates secure random passwords
- User can choose password length
- Uses letters, numbers, and special characters

### 2. Calculator
- Performs basic arithmetic operations
- Addition, Subtraction, Multiplication, and Division

### 3. Rock Paper Scissors Game
- User vs Computer game
- Random computer choice
- Displays winner of each round

## Technologies Used
- Python 3
